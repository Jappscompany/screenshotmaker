Ollama Setup Tutorial

Connect Screenshot (App) Maker to your own Ollama instance for AI translation and text generation.

← Back to Home

What you need

  • A Mac, Windows PC, or Linux machine to run Ollama
  • Ollama installed and running
  • Your iPhone/iPad on the same network as the Ollama machine (recommended)

Step-by-step

  1. Install Ollama: download and install Ollama for your OS.
  2. Start Ollama: run ollama serve (or open the Ollama app on macOS).
  3. Pull a model: run ollama pull llama3.2 (or your preferred model).
  4. Confirm it works: run ollama run llama3.2 and test a prompt.
  5. Find your server address: note your computer's LAN IP (e.g. 192.168.0.10).
  6. Use it in the app: in Screenshot (App) Maker settings, set Host to http://<LAN-IP>:11434.

Notes & troubleshooting

  • If the app cannot connect, check that your phone and computer are on the same Wi‑Fi network.
  • Make sure port 11434 is allowed by your firewall.
  • If you prefer to use a remote server (home PC, VPS), you must expose the port securely (VPN recommended).
